Aqui podes ver os métodos e parametros das requests da api: https://www.weatherapi.com/docs

Os principais métodos são:

API	                    API Method
Current weather	        /current.json or /current.xml
Forecast	            /forecast.json or /forecast.xml
Search or Autocomplete	/search.json or /search.xml
History	                /history.json or /history.xml
Alerts	                /alerts.json or /alerts.xml

Os principais parametros da request são:

Parameter		Description
key	Required	API Key
q	Required	Query parameter based on which data is sent back. It could be following:

-> Esses são parametros obrigatórios, em que key é a api key e q é a query, ou seja
o tipo de consulta que estás a fazer a api saber que conteúdo devolver.
