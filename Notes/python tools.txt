pylint: pylint your_script.py -> checks if your code is well written


bandit: bandit -r your_project/ or bandit script.py -> scans security problems

Nota: para ignora uma linha no bandit é só: # nosec

pytest: pytest tests/test_example.py -> tests your code

# test_math.py

def fib(n):
    if n <= 1:
        return n
    return fib(n-1) + fib(n-2)

def test_fib_zero():
    assert fib(0) == 0

def test_fib_one():
    assert fib(1) == 1

def test_fib_five():
    assert fib(5) == 5
