With poetry:

Basically do this:

#setting new python version
poetry env use 3.8   

#updating dependencies
poetry install

#best command it creates the layout files
poetry new poetry-demo

#installing packages
poetry add package name

#unistalling packages
poetry remove pendulum

#creates virtual env
poetry install ->it will install what it needs to create virtual env and create it 

#activate env
poetry shell

#deativate env
exit
